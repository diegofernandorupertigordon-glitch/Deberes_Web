⚙️ Deber 2 – Funciones

¿Qué es una función?

Una función es un bloque de código que realiza una tarea específica y que puede reutilizarse tantas veces como sea necesario dentro de un programa. Las funciones permiten organizar mejor el código, evitar repetir instrucciones y hacer que los programas sean más fáciles de entender, mantener y modificar.

Gracias a las funciones, un programa puede dividirse en pequeñas partes que resuelven tareas concretas, como calcular valores, mostrar mensajes o procesar datos.

Sintaxis de una función

Una función en JavaScript se define usando la palabra function, seguida del nombre de la función, los parámetros entre paréntesis y el bloque de código entre llaves. 

function nombreFuncion(parametro1, parametro2) {
  // Código que ejecuta la función
  return resultado;
}

Ejemplo de función

Esta función recibe dos números, los suma y devuelve el resultado. 

function sumar(a, b) {
  return a + b;
}

console.log(sumar(5, 3));

Resultado en pantalla:

8

Función sin parámetros

Las funciones también pueden existir sin parámetros y simplemente ejecutar una acción. 

function saludar() {
  console.log("Hola, bienvenido");
}

saludar();

Resultado en pantalla:

Hola, bienvenido

Ejercicios prácticos

Ejercicio 1

Crea una función que reciba un número y devuelva su cuadrado. 

function cuadrado(x){
  return x * x;
}

console.log(cuadrado(4));

Resultado en pantalla:

16

Ejercicio 2

Crea una función que reciba dos números y muestre cuál es el mayor. 

function mayor(a, b) {
  if (a > b) {
    return a;
  } else {
    return b;
  }
}

console.log(mayor(10, 7));

Resultado en pantalla:

10
