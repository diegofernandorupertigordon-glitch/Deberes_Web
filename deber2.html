Ejercicios Prácticos

Uso de sentencias de control

Ejercicio 1 – Verificar si un número es mayor o menor

Este programa verifica si un número es mayor, menor o igual a 10 usando la sentencia if. 

int numero = 12;

if (numero > 10) {
    System.out.println("El número es mayor que 10");
} else if (numero < 10) {
    System.out.println("El número es menor que 10");
} else {
    System.out.println("El número es igual a 10");
}

Ejercicio 2 – Contar del 1 al 5

Este ejemplo usa la sentencia for para mostrar números del 1 al 5. 

for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}
