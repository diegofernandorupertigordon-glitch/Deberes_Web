⚙️ Deber 2 – Funciones

Una función es un bloque de código que realiza una tarea específica y puede reutilizarse tantas veces como sea necesario.

Ejemplo de función

function sumar(a, b) {
  return a + b;
}

console.log(sumar(5, 3));

Ejercicio

Crea una función que reciba un número y devuelva su cuadrado. 

function cuadrado(x){
  return x * x;
}

console.log(cuadrado(4));
