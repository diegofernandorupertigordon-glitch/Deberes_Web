Teoría y práctica

¿Qué son las funciones?

Una función es un bloque de código reutilizable que realiza una tarea específica.

Tipos de funciones

function saludar(){ }
function sumar(a,b){ return a+b; }
function cuadrado(x){ return x*x; }

Ejercicios

function area(base, altura){
 return base * altura;
}

function esPar(n){
 return n % 2 === 0;
}
