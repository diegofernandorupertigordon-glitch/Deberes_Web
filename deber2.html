Funciones en Programación

Teoría, ejemplos y ejercicios prácticos

¿Qué son las funciones?

Una función es un bloque de código que realiza una tarea específica y puede reutilizarse muchas veces. Permite organizar mejor un programa, evitar repetir código y hacer que los programas sean más fáciles de entender.

Tipos de funciones

Función sin parámetros

function saludar(){
    console.log("Hola mundo");
}

Función con parámetros

function sumar(a, b){
    return a + b;
}

Función con retorno

function cuadrado(x){
    return x * x;
}

Ejercicio 1

Crear una función que calcule el área de un rectángulo.

function area(base, altura){
    return base * altura;
}
console.log(area(5, 4));

Ejercicio 2

Crear una función que indique si un número es par.

function esPar(n){
    return n % 2 === 0;
}
console.log(esPar(10));
