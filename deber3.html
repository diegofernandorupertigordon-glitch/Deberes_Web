Programación Orientada a Objetos

¿Qué es la POO?

Es un paradigma de programación que organiza el software en objetos que representan cosas del mundo real.

Los 4 pilares

  • Encapsulamiento: proteger los datos
  • Herencia: reutilizar código
  • Polimorfismo: un mismo método, distintos comportamientos
  • Abstracción: mostrar solo lo importante

Clases y objetos

class Persona {
    constructor(nombre){
        this.nombre = nombre;
    }
    saludar(){
        return "Hola " + this.nombre;
    }
}
let p = new Persona("Diego");
