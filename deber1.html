Sentencias de Control

Desarrollo Web – Fundamentos de Programación

¿Qué son las sentencias de control?

Las sentencias de control son estructuras que permiten dirigir el flujo de ejecución de un programa. Gracias a ellas, un programa puede tomar decisiones, repetir instrucciones o ejecutar bloques de código dependiendo de ciertas condiciones.

En programación web, estas sentencias permiten que una página reaccione de forma dinámica según la información que el usuario ingresa, como por ejemplo validar un formulario o mostrar mensajes distintos según una condición.

Tipos de sentencias de control

1. if / else

Permite ejecutar un bloque de código si una condición es verdadera y otro si es falsa. 

if (condicion) {
    // se ejecuta si es verdadera
} else {
    // se ejecuta si es falsa
}

2. switch

Se utiliza cuando existen múltiples opciones posibles para una misma variable. 

switch (variable) {
    case valor1:
        // instrucciones
        break;
    case valor2:
        // instrucciones
        break;
    default:
        // si no coincide
}

3. while

Repite un bloque de código mientras una condición sea verdadera. 

while (condicion) {
    // instrucciones
}

4. for

Se utiliza cuando se conoce cuántas veces se repetirá una acción. 

for (let i = 0; i < 5; i++) {
    // instrucciones
}

Ejercicios prácticos

Ejercicio 1: Verificar si una persona es mayor de edad

En este ejercicio usamos una sentencia if para determinar si una persona puede ingresar según su edad. 

let edad = 20;

if (edad >= 18) {
    console.log("Puede ingresar");
} else {
    console.log("No puede ingresar");
}

Si la edad es 18 o mayor, el programa mostrará “Puede ingresar”. Caso contrario mostrará “No puede ingresar”.

Ejercicio 2: Mostrar los números del 1 al 5

En este ejemplo utilizamos una sentencia for para repetir una acción varias veces. 

for (let i = 1; i <= 5; i++) {
    console.log(i);
}

El ciclo se repite cinco veces y muestra los números del 1 al 5 en la consola.
