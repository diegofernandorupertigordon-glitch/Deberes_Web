Sentencias de Control

Conceptos y Sintaxis

¿Qué son las sentencias de control?

Las sentencias de control son estructuras que permiten que un programa tome decisiones y controle el flujo de ejecución. Gracias a ellas, el programa puede ejecutar diferentes acciones según una condición o repetir instrucciones varias veces.

Tipos de sentencias de control

1. If

Se usa para ejecutar un bloque de código si una condición es verdadera.

if (condicion) {
    // código a ejecutar
}

2. If – Else

Permite ejecutar una acción si la condición es verdadera y otra si es falsa.

if (condicion) {
    // código si es verdadero
} else {
    // código si es falso
}

3. Switch

Se utiliza cuando hay muchas opciones posibles.

switch (opcion) {
    case 1:
        break;
    case 2:
        break;
    default:
        break;
}

4. For

Sirve para repetir código una cantidad definida de veces.

for (int i = 0; i < 5; i++) {
    // código que se repite
}

5. While

Repite el código mientras una condición sea verdadera.

while (condicion) {
    // código que se repite
}
