Deber 1: Sentencias de Control

1. ¿Qué son?

Permiten controlar el flujo de ejecución de un programa según condiciones o repeticiones.

2. Ejemplos en JavaScript:

let edad = 18;
if (edad >= 18) {
    console.log('Mayor de edad');
} else {
    console.log('Menor de edad');
}

for (let i = 1; i <= 5; i++) {
    console.log('Número ' + i);
}
